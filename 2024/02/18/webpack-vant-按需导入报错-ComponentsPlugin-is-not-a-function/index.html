webpack-vant-按需导入报错-ComponentsPlugin-is-not-a-function

发表于 2024-02-18 | 分类于 Vant | 阅读次数:
字数统计: 223

背景

使用 Vant+vue-cli的学习项目中，想使用 `unplugin-vue-components@0.26.0,实现按需导入 报错TypeError: ComponentsPlugin is not a function`

解决方案

官方文档已经说明了，官方文档
相关 issue, [文档 unplugin-vue-components@0.26.0 引用方式有误 #12499]
(https://github.com/youzan/vant/issues/12499)

请注意你项目 unplugin-vue-components 的版本，如果大于等于 0.26.0 你需要注释第一行，反之，则相反
